Introduction

This project aims to build a model that incorporate deep learning techniques together with natural language processing (NLP) to analyse the social media discussion messages with the purpose to identify topic automatically and determine the trend of the discussion

BERT Model

BERT model will be used to achieve the goals. It is a state-of-art NLP model

Topic Identification

Identify the trend of discussion of a day

Sentiment Analysis

Analyze the public sentiment on the topics
